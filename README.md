# GUI：学生工作绩效材料自动生成工具

本项目为学生工作数字化辅助工具，支持从新闻稿或推文自动生成绩效考核材料。

## 功能
- 批量读取 `.txt / .docx / .doc / .pdf`
- 自动提取时间、地点、主办单位
- 自动匹配考核指标（1.1~5.1）
- 一键生成带页眉页脚的 `.docx` 模板
